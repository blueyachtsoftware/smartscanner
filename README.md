# スマート スキャナー

スマート スキャナー は、以下の Windows 向けアプリケーションと連携可能な Android スマートフォン向けアプリです。

- [バーコード マネージャー for Windows](https://apps.microsoft.com/detail/9n9pzsp6wc3k)
- [バーコード マネージャー Pro for Windows](https://apps.microsoft.com/detail/9NS374KT3QL9)
- [バーコード マネージャー for Business](https://apps.microsoft.com/detail/9pf2kpxk63nm)
- [バーコード マネージャー for Enterprise](https://apps.microsoft.com/detail/9p2vx7gsj4tw)

このアプリを使用することで、スマートフォンのカメラを利用してバーコードをスキャンし、Windows アプリと連携してデータを送信することができます。

## 主な機能

- Android端末でのバーコードスキャン
- Windowsアプリとのリアルタイム連携
- シンプルで直感的な操作画面
- 商用利用にも対応（条件あり）

## インストール方法（Android 9.0以降）

1. [リリースページ](https://github.com/blueyachtsoftware/smartscanner/releases)からAPKファイルをダウンロードします。
2. ダウンロード後、ブラウザに対して「この提供元からのインストールを許可」を一時的に有効化します。
3. アプリをインストールします。
4. インストール完了後、許可設定を元に戻します。（推奨）
5. アプリを起動し、Windowsアプリと接続設定を行います。
